# Pixel-Art: Image Mosaic Project

## Overview

This project demonstrates the creative use of image processing techniques to transform a standard image into a visually striking mosaic. Each pixel of the original image is replaced with a unique image, preserving the essence of the original while creating a dynamic artwork. This project showcases both my technical skills and creative vision in the field of image processing.

## Technologies Used

- **Pandas**: For data manipulation and analysis.
- **NumPy**: For numerical operations on arrays.
- **Scikit-learn**: For any machine learning tasks.
- **OpenCV**: For image processing operations.
- **Pillow**: For opening, manipulating, and saving image files.
